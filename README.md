# mathmini-choco
Sample math library containing add and sub functions

# Installation
```
choco install mathmini --version=1.0.0
```

# Usage
```
mathmini add 10 5
mathmini sub 10 5
```

# Release and Publish
1. Update `version` in `mathmini.nuspec`
2. Commit changes
3. Run `git tag v1.0.0`
4. Run `git push origin v1.0.0`
